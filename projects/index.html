A Covid-19 Test Tracker

A web-scraper to track the number of Coronavirus tests performed by 33 different countries. I publish daily raw results and weekly graphs. Written in Python with Selenium and BeautifulSoup4 + R for the Graphs.

A Study of Twitter

Over a series of weeks I scraped around 50,000 tweets along with the information from 600 of the most important accounts on political twitter to understand the makeup of British political Twitter. Uses Gephi, Plotly-Dash, Plotly-JS and Bootstrap for visualisation. Tweepy + Twint (python) were used for data collection and Fasttext (Python) for AI Sentiment Analysis.

Chexit.co.uk

A website to test the Brexit positions of all MPs (before 2019 December election) using over 50 different parliament votes on Brexit related issues. It was written in PHP (with MySQL as database) with Bootsrap, Chart-Js and Jquery on the front-end.

The Monocle

An online only news source that I created for Durham University along with some friends. It won best development at the Northern Region Student Publication Awards and is currently nominated for 3 national awards.

CheapShot

CheapShot is a tool that uses Machine Learning and Natural Language Processing to identify potentially offensive tweets from someones twitter account.
